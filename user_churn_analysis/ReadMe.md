模型结果评价：

精确率 = TP/(TP+FP) ：在预测为流失的用户中，预测正确的（实际也是流失）用户占比
召回率 = TP/(TP+FN) : 在实际为流失的用户中，预测正确的（预测为流失的）用户占比
F1值为精确率和召回率的调和均值，相当于这两个的综合评价指标。
